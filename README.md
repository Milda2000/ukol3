Program slouží k výpočtu vzdáleností mezi adresními místy z databáze OSM kontejnery tříděného odpadu. Pro fungování programu je potřebné mít tato data ve formě geojson uložena v souborech adresy.geojson, respektivě kontejnery.geojson.
Program následně z těchto dat spočítá nejkratší průměrné vzdálenosti. Medián vzdáleností a nejdelší minimální vzdálenost ke kontejneru.
